# Math.abs 메서드

- 숫자의 절대값을 반환한다.
  - 따라서 결과값은 항상 양수 아니면 0이다.
- NaN : 값이 숫자가 아닐 경우

<pre>
    <code>
        Math.abs(-1) // 1
        Math.abs(1) // 1
        Math.abs(0) // 0
    </code>
</pre>

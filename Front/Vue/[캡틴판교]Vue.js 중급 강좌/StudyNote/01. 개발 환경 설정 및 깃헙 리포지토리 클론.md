# 강의 소개

### ✔ 강의에서 다루는 내용

1. Vue CLI를 이용한 **프로젝트 구성** 방법
2. **컴포넌트 기반 설계** 방법
3. **컴포넌트 구조화** 및 **컴포넌트 통신** 방법
4. **ES6**를 이용한 효율적이고 간결한 뷰 코딩 방법 ✨
5. **Vuex**를 이용한 상태 관리의 이해 및 적용 방법 ✨

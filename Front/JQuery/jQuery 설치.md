# jQuery 로컬 다운로드

:link: [jQuery](https://jquery.com/)

1. Download jQuery
2. Download the compressed, production jQuery 3.6.1 클릭
3. 다른 이름으로 저장
4. 사용하고자 하는 프로젝트 dir에 이동
5. html의 head태그에서 src 지정

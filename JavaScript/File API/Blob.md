# Bolb (Binary Large Object)

- 이미지, 사운드, 비디오와 같은 멀티미디어 데이터를 다룰 때 사용한다.
  - 즉, JavaScript 네이티브 형식이 아닌 데이터를 나타낼 수 있다.
- 데이터의 크기(Byte) 및 MIME 타입을 알아내거나, 데이터를 송수신을 위한 작은 Bolb 객체로 나누는 등의 작업에 주로 사용된다.

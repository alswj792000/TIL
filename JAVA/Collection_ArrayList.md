------
List 계열
: 자료들을 순차적으로 나열한 자료구조<br>
	인덱스(순서유지 가능)으로 관리하고, 중복해서 객체 저장을 할 수 있다.<br>
	
ArrayList
: List의 후손으로 초기 저장용량은 10으로 자동 설정된다. <br>
		저장용량을 초과한 객체들이 들어올 경우 용량이 자동으로 증가한다. <br>
		동기화(Syncronized)를 제공하지 않는다.<br>

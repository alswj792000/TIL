# align-items

- 컨테이너 내부 항목의 기본 정렬을 지정
- 세로축을 기준으로 정렬을 한다.

|                  |                                    |
| ---------------- | ---------------------------------- |
| flex-start       | 컨테이너의 최상단으로 정렬         |
| flex-end         | 컨테이너의 최하단으로 정렬         |
| center           | 컨테이너의 세로 축의 중앙으로 정렬 |
| baseline         | 컨테이너의 시작위치에 정렬         |
| stretch(default) | 컨테이너에 맞게 늘림               |
